# lab-1-circuit-playground-classic

La finalidad de este laboratorio es familiarizarse con el **IDE de Arduino** para la programación y el uso de la plataforma de desarrollo **Circuit Playground Classic**.

Siga los siguientes enlaces para que conozca qué sensores brinda la plataforma de desarrollo y cómo programar las luces:

[Introducing Circuit Playground](https://learn.adafruit.com/introducing-circuit-playground/overview)

[Circuit Playground Bike Light](https://learn.adafruit.com/circuit-playground-bike-light/overview)

Con base en esto, investigue, diseñe y programe el tipo de **semáforo de control basado en tiempo fijo**.

Debe generar un documento donde evidencie lo siguiente:

- Conocimiento sobre la plataforma de desarrollo utilizada
- Programación de las luces según los enlaces brindados
- Conocimiento sobre el semáforo de control basado en tiempo fijo
- Diseño e implementación del semáforo de control basado en tiempo fijo
